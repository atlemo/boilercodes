What's this?

A designer learning to code using Flex and Grid ended up as this, Boiler Codes.

Not a new framework, rather a small selection of starting point templates and experiments for modern browsers. Plain HTML and CSS. Meant to give you a head start for your next web project, or to pick apart and see how it's made. See something that could be done better? - Please, contribute →

No fuzz, no prefix. All responsive, obviously. Updated infrequently.

Free to use, forever.
